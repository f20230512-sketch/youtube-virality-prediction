# YouTube Virality Prediction & Engagement Segmentation

End-to-end machine learning pipeline for predicting YouTube video virality using NLP, clustering, and XGBoost.

## Tech Stack
- Python
- SQL
- scikit-learn
- XGBoost
- SHAP
- Pandas
- Matplotlib

## Features
- NLP feature engineering from titles and tags
- K-Means audience segmentation
- XGBoost virality prediction
- ROC/AUC evaluation
- SHAP explainability analysis
- Trending video clustering

## Results
- 82%+ AUC score
- 200K+ engagement records analyzed
- 4 content archetypes identified
- Viral prediction pipeline with explainable ML outputs
